## game_for_Lesta

реализована простая текстовая RPG (без автобоя)

Для улучшения игрового опыта добавлено:
1. в первой комнате всегда нет врага, а в последней он всегда есть.
2. наличие противников в комнате 2, 3, 4 подбирается рандомно.
3. если в комнате есть противник, то игрок может выбрать атаку или другие действия.
4. если выбирает другие действия - теряет жизни(
